This folder contains the notebook 'data_analysis.ipynb' which contains all the analysis performed after extraction of mean voxel intensities from the ROI's.
This notebook includes, 
* Normalization of mean intensities from SNpc, LC and putamen on WM intensities
* Merging of the MRI data into one combined MRI dataset
* Merging MRI data with clinical data into one total dataset
* Statistical tests performed (group comparisons between two groups (PD vs HC, PIGD vs TD), laterality assessment, multiple group comparisons (HC vs PIGD vs TD vs Indeterminate), effect size calculation)
* Generalized Linear Model (GLM) analysis
* Code for all plots presented in the results chapter
